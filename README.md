Iris Classification using Decision Tree with Streamlit
Overview

This project is a machine learning web application that classifies Iris flowers into three species using a Decision Tree Classifier. The application is built with Python, Scikit-learn, and Streamlit for an interactive frontend.

The model predicts the Iris flower species based on the following input features:

Sepal Length
Sepal Width
Petal Length
Petal Width
Features
Train a Decision Tree model on the Iris dataset
Interactive Streamlit web interface
Real-time predictions
Simple and beginner-friendly project structure
Visualization of model accuracy and predictions

Python
Scikit-learn
Pandas
NumPy
